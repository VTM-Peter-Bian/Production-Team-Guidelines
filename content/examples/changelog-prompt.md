# VTM CHANGELOG 規範

**版本：** v0.1.0

---

請查看本次變更內容，在 `CHANGELOG.md` 頂部新增對應版本區塊。

## 任務

1. 檢視本次修改的檔案與內容
2. 確認版本號（對照版本號規範）
3. 輸出可直接貼入 `CHANGELOG.md` 頂部的新版本區塊

## 撰寫規則

- 最新版本置頂，不修改歷史記錄
- 分組：`Added` / `Changed` / `Fixed`
- 每條格式：`- 【分類名】變更描述（含 `路徑`）`
- 只記錄重大變更，描述須具體

## 區塊格式

```markdown
## [1.2.0] - 2026-03-15

### Added
- 【模組名】新增 `路徑/檔案` …

### Changed
- 【模組名】調整 `路徑/檔案` …

### Fixed
- 【模組名】修復 `路徑/檔案` …
```

## 輸出格式

```
（輸出完整版本區塊，可直接貼至 CHANGELOG.md 最上方）

---
請確認版本號無誤後，手動貼入 CHANGELOG.md 並提交。
```
